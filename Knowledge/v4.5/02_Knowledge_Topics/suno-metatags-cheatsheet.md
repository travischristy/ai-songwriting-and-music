# Suno AI Metatags Cheatsheet

## Metatag Formatting Rules

1. Always use square brackets: `[Section: Modifier]`
2. Keep metatags simple (3 words max in modifiers)
3. Place at the beginning of sections
4. Never use metatags mid-line
5. Character limit: 200 characters for style input

## Section Structure Metatags

| **Section Type** | **Metatag Options**                                | **Purpose**                  |
| ---------------- | -------------------------------------------------- | ---------------------------- |
| Intro            | `[Intro]`<br/>[In`tro: Instrumental]<br/>[I`ntro`` | Establishes song's beginning |
| Verse            | `[Verse]`<br/>[Ve`rse 1]<br/>[V`erse``             | Main narrative sections      |
| Pre-Chorus       | `[Pre-Chorus]`<br/>[Pr``                           | Builds tension before chorus |
| Chorus           | `[Chorus]`<br/>[Ch`orus: Energetic]<br/>[C`horu``  | Main hook section            |
| Post-Chorus      | `[Post-Chorus]`<br/>[Po``                          | Extension after chorus       |
| Bridge           | `[Bridge]`<br/>[Br``                               | Provides contrast/shift      |
| Outro            | `[Outro]`<br/>[Ou``                                | Concludes the song           |
| Hook             | `[Hook]`                                           | Emphasizes catchy section    |
| Break            | `[Break]`<br/>[In``                                | Instrumental section         |

## Most Effective Combinations

### For Pop Songs

1. `[Verse: Intimate Vocals]`
2. `[Pre-Chorus: Building Energy]`
3. `[Chorus: Catchy Hook]`
4. `[Post-Chorus: Danceable]`
5. `[Bridge: Emotional Shift]`

### For R&B/Soul

## Implementation Tips

1. **Clarity Over Complexity**: Simple, clear metatags work better than complex ones
2. **Consistent Tagging**: Tag all major sections for better song structure
3. **Complementary Tags**: Ensure metatags work with your style input
4. **Strategic Placement**: Place most important metatags in chorus and bridge
5. **Avoid Redundancy**: Don't repeat what's already in your style input
6. **Test Variations**: If a metatag doesn't work, try alternatives

## Common Mistakes to Avoid

1. **Over-tagging**: Too many metatags can confuse the AI
2. **Inconsistent Formatting**: Maintain consistent [Section: Modifier] format
3. **Contradictory Instructions**: Don't request conflicting elements
4. **Unclear Boundaries**: Clearly separate sections with metatags
5. **Ignoring Character Limits**: Stay within 200 characters for style input
6. **Generic Tags**: More specific tags typically work better

## Advanced Usage

1. **Emotional Arc**: Use metatags to create emotional progression
2. **Transition Effects**: Tag transitions between sections
3. **Vocal Direction**: Specify exact vocal qualities
4. **Production Elements**: Suggest specific production techniques
5. **Section Variations**: Differentiate variations of the same section type
